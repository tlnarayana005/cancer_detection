# Contributing to Cancer Detection AI

Thank you for your interest in contributing to this open source cancer detection project.

## How to contribute

1. Fork the repository.
2. Create a feature branch: `git checkout -b feature/your-description`.
3. Commit your changes: `git commit -m "Add feature or fix bug"`.
4. Push to your branch: `git push origin feature/your-description`.
5. Open a pull request with a clear description.

## What is welcome

- Bug fixes
- Improved documentation
- New model backbones and explainability methods
- Better dataset handling and preprocessing
- Performance and reliability improvements

## Development workflow

- Keep code modular and maintainable
- Add tests for new behavior
- Use consistent naming and comments
- Share reproducible training results
